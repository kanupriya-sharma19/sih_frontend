

# 🚆 DRISHTI

🚀 AI-Powered Train Traffic Optimization & Decision Support System

AutoGen • Graph Optimization • Ant Colony • Genetic Algorithms • Neo4j • MongoDB • SUMO • Flask

---

## 📌 Overview

**DRISHTI (Decision & Routing Intelligence System for Handling Train Information)** is an intelligent AI-driven platform designed to automate and optimize train traffic management.

It addresses real-world railway challenges such as **manual scheduling, congestion, delays, and inefficient decision-making** by leveraging advanced optimization algorithms, simulation, and real-time data processing.

The system enables **faster, safer, and more efficient train operations** through intelligent routing, predictive analytics, and decision support tools.

---

## ⚡ Key Features

### 🤖 Intelligent Decision Support

* AI-powered assistance for section controllers
* Real-time decision-making for delays & disruptions
* Human-in-the-loop system for reliability

### 🧠 Advanced Optimization Engine

* Graph-based modeling of stations, tracks & schedules
* Conflict-free routing using signals & rerouting logic
* Ant Colony Optimization for adaptive routing
* Genetic Algorithms for global timetable optimization
* Simulated agents for real-time disruption handling

### 🔁 Digital Twin Simulation

* Real-time system simulation for testing scenarios
* “What-if” analysis for accidents, weather, and peak loads
* Enables safe experimentation without real-world risk

### 🕸️ Smart Data & Analytics

* Train delay trend analysis
* Time vs distance visualization
* Station occupancy tracking
* Throughput efficiency metrics
* Self-monitoring alerts for incorrect decisions

### 🔎 Real-Time Integration

* Live updates via government APIs
* Seamless interaction with railway data systems
* Centralized dashboard for monitoring & control
